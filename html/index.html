scitags

Maintains map of registered scitags values, see Scientific network tags (scitags)

map:[bits:ip6flowlabel:261884]:scitag

Use the above expression in flow definitions to extract and decode registered scitag values from IPv6 flow label

map:ip6scitag:scitag

Use the above expression in flow definitions decode registered scitag values from IPv6 header extension
